Escape From Darkness (Game Jam)

21 May 2021

It is a Run and Jump style game that I made the music and visuals myself in 3 days in the Hackaton Isparta Competition.

During the 3-day period, I understood the importance of time planning in game programming. With the color mechanics I developed, I learned the using different layers, and I realized that different level designs can be created with the flexibility of color mechanic.

Gameplay

The player tries to escape from the darkness on different platforms by using the colors red, green and blue. The player can pass through platforms of the same color as himself.
